# tmux

```
Source: https://github.com/tmux/tmux/wiki
Description: Run & manage multiple terminal sessions inside 1 terminal window

narnia0@narnia:~$ tmux --help
usage: tmux [-2CDlNuVv] [-c shell-command] [-f file] [-L socket-name]
            [-S socket-path] [-T features] [command [flags]]
narnia0@narnia:~$ man tmux

# Vertical split
Ctrl + B + % (hold shift 5)

# Horizontal split
Ctrl + B + " (hold shift ')

# Switch between panes
Ctrl + B + UP
Ctrl + B + DOWN
Ctrl + B + LEFT
Ctrl + B + RIGHT

# Resize panes
Ctrl + B + Alt + UP
Ctrl + B + Alt + DOWN
Ctrl + B + Alt + LEFT
Ctrl + B + Alt + RIGHT
```
